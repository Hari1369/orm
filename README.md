# orm

| CRUD Action       | HTTP Method | URL Example       | What it does       |
| ----------------- | ----------- | ----------------- | ------------------ |
| List              | GET         | `/products/`      | Show all records   |
| Create            | POST        | `/products/`      | Add a new record   |
| Detail (Retrieve) | GET         | `/products/<id>/` | Show single record |
| Update            | PUT/PATCH   | `/products/<id>/` | Update data        |
| Delete            | DELETE      | `/products/<id>/` | Remove record      |
